Best practices when using classes vs. ids

A HTML & CSS concept

June 21, 2015

The benefits of using classes and ids at all

Classes and ids are attributes that help control the structure and layout of a page. If you want to group together related elements on a page you can make use of a HTML element (e.g. <div>) together with class or id attributes. This will indicate the type of content you will find in these groups and allow you to style this group differently using CSS. Using these attributes will only affect their appearance on the page if combined with a CSS rule with special styling.


When do I use classes vs. ids then?

An id is uniquely identifying one element within your code whereas a class can be attributed to several elements in your code. If you want to style one element differently from any other instance of that element, like for example a <h1> heading, you would use an id. Only this one heading that you associate with the special id will be displayed in the special format that you define in you CSS file. It could also be the case that you want to style several elements as being different from the other elements on your page. In that case you would use the class attribute. For example, in this blog post I used a class “bold” for all bold words within a text paragraph together with the element <span>. I want to display several words in bold, which means that I want to style several elements in the same way. Therefore, I use the class attribute instead of the id attribute, because I can assign it to several elements in my code.


How do I use classes and ids in my code?

I will explain how to incorporate the class or id attributes in your code together with the <span> element.


Class:

  • In the HTML code you assign elements with a class attribute in the following way: <span class="">, for example <span class="bold">.
  • In the CSS code a class attribute is indicated as a CSS selector by a period symbol followed by the class attribue value. In the above example: .bold {text-decoration:bold}

Id:

  • In the HTML code you assign one element one unique id in the following way: <span id="">, for example <span class="123">.
  • In the CSS code the unique id attribute is indicated as a CSS selector by a hash symbol followed by the id attribue value. In the above example: #123 {}
