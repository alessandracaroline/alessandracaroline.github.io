RegEx - Regular Expressions

What they do and how they help you

August 09, 2015

What is a regular expression?

A regular expression (short: RegEx) is a sequence of characters that define a search pattern. They are useful when searching for data in text or when performing "search-and-replace" operations by matching this search pattern in the text. A RegEx can be a single character, or a more complicated sequence of characters. RegEx can make a search much more powerful, because they can include modifiers and for example perform a case insensitive search.


Syntax

/pattern/modifiers;


A good reference on RegEx syntax can be found under this link: w3schools' JavaScript RegExp Reference


Modifiers

Modifier Description
i perform case-insensitive match
g perform a global match (finds all matches, does not stop after the first match)
m perform multiline matching


Brackets

Brackets are used to find a range of characters

Expression Description
[abc] find any character between the brackets
^[abc] find any character NOT between the brackets
[a-z] or [A-Z] or [A-z] find any character from lowercase a to lowercase z / from uppercase A to uppercase Z / from uppercase A to lowercase Z
[0-9] find any digit between the brackets
(a|b) find any of the alternatives


Metacharacters

Metacharacters have a special meaning. They are written with a backslash. Using a capital letter will negate the meaning of the lowercase letter.

Character Description
. find a single character, except newline or line terminator
\w find a word character
\d find a digit
\s find a whitespace character
\b find a match at the beginning/end of a word


Quantifiers

Quantifiers define quantities.

Character Description
n+ matches any string that contains at least one 'n'
n* matches any string that contains zero or more occurrences of 'n'
n? matches any string that contains zero or one occurence of 'n'
{a} matches any string that contains exactly a occurences of the pattern
{a, b} matches any string that contains exactly a to b occurences of the pattern
{a,} matches any string that contains at least a occurences of the pattern
\b find a match at the beginning/end of a word


Application in JavaScript

In JavaScript, regular expressions are often used with the string methods:

  • match(), which searches for a match of the pattern and returns the pattern that match.
  • search(), which searches for a match of the pattern and returns the position of the match.
  • replace(), which returns a modified string where the pattern is replaced.


Examples

RegEx Example Description
/us/ matches "us" in "us", "house", "mouse", "fuss".
/^un/ matches "un" in the beginning of the expression that is being matched, e.g. "unfair".
/ab*c/ matches "ac", "abc", "abbc" and "abbbbbbbc" the last character before the *, can have zero or more occurences.
/ab+c/ matches "abc", "abbc" and "abbbbbbbc", but not "ac" the last character before the *, has at least one occurence.
